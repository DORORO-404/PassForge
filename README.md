# passforge 🔐

A simple password generator tool built for Kali Linux (and other Linux systems).

## ✨ Features

- Generates strong and random passwords
- Supports custom password length
- Option to include:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- Lightweight, fast, and terminal-friendly

## 🧪 Example Usage
